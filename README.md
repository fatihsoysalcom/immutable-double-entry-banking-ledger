# Immutable Double Entry Banking Ledger

This example demonstrates a secure, immutable double-entry ledger system with idempotency guarantees, moving beyond simple CRUD operations. It uses SQLite in-memory to showcase atomic transactions, balance calculations from ledger history, and duplicate request prevention.

## Language

`python`

## How to Run

Run 'python ledger_api.py' in your terminal. No external dependencies are required.
